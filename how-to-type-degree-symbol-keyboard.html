How to Type Degree Symbol on Keyboard

Complete guide with keyboard shortcuts and Alt codes for all operating systems

Windows, Mac, Linux, and mobile device shortcuts included

Quick Reference

Platform Shortcut Symbol Notes
Windows Alt + 0176 ° Use numeric keypad
Mac Option + Shift + 8 ° Also: Option + K
Linux Ctrl + Shift + U 00B0 ° Unicode input method
iPhone/iPad Hold 0 key ° Touch keyboard

Windows Keyboard Shortcuts

Alt Code Method

Basic Degree Symbol

Alt + 0176
  1. 1. Make sure Num Lock is ON
  2. 2. Hold down the Alt key
  3. 3. Type 0176 on the numeric keypad
  4. 4. Release Alt key

Celsius Symbol

Alt + 8451

Types the degree Celsius symbol directly

Character Map

Using Character Map

  1. 1. Press Windows Key + R
  2. 2. Type charmap and press Enter
  3. 3. Find the degree symbol (°)
  4. 4. Double-click to select it
  5. 5. Click Copy and paste where needed

Pro Tip:

In Character Map, check "Advanced view" and search for "degree" to find all degree-related symbols quickly.

Mac Keyboard Shortcuts

Option Key Shortcuts

Primary Method

Option + Shift + 8

This is the most common and reliable method

Alternative Method

Option + K

Also produces the degree symbol

Character Viewer

Quick Access

Control + Command + Space
  1. 1. Press the shortcut to open Character Viewer
  2. 2. Search for "degree"
  3. 3. Double-click the symbol you want

Linux Keyboard Shortcuts

Unicode Input Method

GTK/GNOME Systems

Ctrl + Shift + U 00B0

Type the code, then press Enter

KDE Systems

Alt + . then 00B0

Compose key method

Copy-Paste Method

The universal method that works on all systems:

°

Copy this symbol and paste it where needed. You can also use:

℃ ℉

Mobile Device Methods

iPhone/iPad

Hold Method

Tap and hold the 0 key on the keyboard, then select ° from the popup menu.

Text Replacement

Set up text replacement in Settings > General > Keyboard > Text Replacement.

Android

Symbols Keyboard

Switch to the symbols keyboard (?123) and look for the degree symbol.

Gboard Method

Long-press the comma (,) key to access additional symbols including °.

Application-Specific Shortcuts

Microsoft Office

Word/Excel: Insert > Symbol > More Symbols

Shortcut: Ctrl + @, Space

AutoCorrect: (c) becomes °C

Google Docs

Insert: Insert > Special characters

Search: Type "degree" in search box

Keyboard: Platform shortcuts work

Programming

HTML: &deg; or &#176;

JavaScript: "\u00B0"

Python: "\u00B0" or chr(176)

Troubleshooting Common Issues

Alt Codes Not Working?

  • Check Num Lock is enabled
  • Use the numeric keypad, not number row
  • Try different keyboard layouts

General Solutions

  • Copy and paste from this website
  • Use Character Map or Character Viewer
  • Set up text replacement for frequent use
