How to Type Degree Symbol on Mac

Complete guide for macOS users with multiple methods and shortcuts

Updated for macOS Sonoma and later versions

Quick Methods

Keyboard Shortcut

Option + Shift + 8

The fastest way to type the degree symbol on any Mac keyboard.

Pro Tip:

For Celsius (°C), type Option + Shift + 8, then C. For Fahrenheit (°F), type Option + Shift + 8, then F.

Character Viewer

Control + Command + Space

Access the full character palette including degree symbols and variations.

Steps:

  1. 1. Press Control + Command + Space
  2. 2. Search for "degree"
  3. 3. Double-click the symbol you want

Detailed Methods

Method 1: Keyboard Shortcuts

Basic Degree Symbol (°)

Option + Shift + 8

This works on all Mac keyboards, including laptop keyboards.

Alternative: Option + K

Option + K

This also produces the degree symbol and is easier to reach on some keyboards.

Keyboard Layout Note

These shortcuts work regardless of your keyboard layout (US, UK, International, etc.).

Method 2: Character Viewer

Step-by-Step Instructions

  1. 1 Press Control + Command + Space to open Character Viewer
  2. 2 Click the search bar and type "degree"
  3. 3 Select your desired degree symbol from the results
  4. 4 Double-click to insert it into your document

Character Viewer Features:

  • • Access to all degree symbol variations
  • • Recently used symbols for quick access
  • • Favorite symbols for frequent use
  • • Works in all macOS applications

Method 3: Emoji & Symbols Menu

Using the Menu Bar

  1. 1 Click the input menu in the menu bar (looks like a flag or character)
  2. 2 Select "Show Emoji & Symbols"
  3. 3 Navigate to the degree symbols section

Application-Specific Methods

Pages and Keynote

Method 1: Use the keyboard shortcut Option + Shift + 8

Method 2: Go to Edit > Emoji & Symbols

Method 3: Use the Format menu to insert special characters

Numbers

Method 1: Keyboard shortcuts work in cells

Method 2: Use the Function menu > Insert > Symbol

Method 3: Format cells to include degree symbols automatically

Troubleshooting

Shortcuts Not Working?

  • Check if you're using the correct key combination
  • Make sure no other apps are interfering with keyboard shortcuts
  • Try using Character Viewer as an alternative method

Still Having Issues?

Check your keyboard settings in System Preferences > Keyboard > Input Sources to ensure the correct layout is selected.
