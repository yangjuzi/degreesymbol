How to Type Degree Symbol on iPhone

Complete guide for iOS users with keyboard shortcuts and text replacement

Works on iPhone and iPad running iOS 15 or later

Quick Methods

Zero Key Method

Hold down the 0 key

The fastest way to access degree symbol on iOS keyboard.

Steps:

  1. 1. Open the keyboard
  2. 2. Tap and hold the 0 key
  3. 3. Select ° from the popup

Text Replacement

Set up custom shortcuts

Create your own shortcuts for frequent degree symbol use.

Example:

Type "deg" to automatically insert ° symbol

Detailed Methods

Method 1: Keyboard Hold Method

Step-by-Step Instructions

  1. 1 Open any app where you can type (Messages, Notes, Mail, etc.)
  2. 2 Tap to bring up the iOS keyboard
  3. 3 Find and tap the 0 (zero) key on the number keyboard
  4. 4 Hold your finger down on the 0 key
  5. 5 A popup menu will appear with the degree symbol (°)
  6. 6 Slide your finger to the degree symbol and release

Pro Tip:

This method works in most iOS apps including Messages, Notes, Mail, Safari, and third-party apps.

Method 2: Text Replacement Setup

Setting Up Text Replacement

  1. 1 Open Settings app on your iPhone
  2. 2 Tap General
  3. 3 Tap Keyboard
  4. 4 Tap Text Replacement
  5. 5 Tap the + button in the top-right corner
  6. 6 In the Phrase field, type or paste the degree symbol (°)
  7. 7 In the Shortcut field, type a short code like "deg" or "degree"
  8. 8 Tap Save in the top-right corner

Usage:

Now when you type your shortcut (e.g., "deg"), it will automatically be replaced with the degree symbol (°).

Method 3: Voice Dictation

Using Siri Dictation

  1. 1 Tap the microphone icon on the keyboard
  2. 2 Say "degree symbol" or "degrees" out loud
  3. 3 iOS will insert the degree symbol (°)

Voice Commands:

  • • "degree symbol" → °
  • • "degrees" → °
  • • "degree Celsius" → °C
  • • "degree Fahrenheit" → °F

Troubleshooting

Degree Symbol Not Available?

  • Make sure you're holding the 0 key long enough
  • Try switching to the number keyboard if you're on the letter keyboard
  • Check if your iOS version supports this feature (iOS 9+)

Text Replacement Not Working?

  • Make sure Text Replacement is enabled in Settings
  • Restart your device if the issue persists
